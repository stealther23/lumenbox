# lumenbox

![npm](https://img.shields.io/npm/dt/lumenbox.svg?style=for-the-badge)

Javascript lightweight library for image overlays. No jQuery dependencies.

## Installation
```npm i lumenbox```

## Usage
1. Add the data attribute `data-lumenbox` to your photos.
